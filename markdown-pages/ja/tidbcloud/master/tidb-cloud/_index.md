# TiDB Cloudのドキュメント {#tidb-cloud-documentation}

[TiDB Cloud](https://pingcap.com/products/tidbcloud)は、完全に管理されたサービスとしてのデータベース (DBaaS) であり、TiDB のすべての優れた機能をクラウドにもたらし、データベースの複雑さではなく、アプリケーションに集中できるようにします。
