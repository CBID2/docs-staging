# ロールを削除 {#drop-role}

このステートメントは、以前に`CREATE ROLE`で作成されたロールを削除します。

## あらすじ {#synopsis}

```ebnf+diagram
DropRoleStmt ::=
    'DROP' 'ROLE' ( 'IF' 'EXISTS' )? RolenameList

RolenameList ::=
    Rolename ( ',' Rolename )*
```

mysql> GRANT analyticsteam TO jennifer;
Query OK, 0 rows affected (0.01 sec)

## MySQL の互換性 {#mysql-compatibility}
